# Code of Conduct

## Our Standards

We are committed to a collaborative and professional engineering environment.
Participants are expected to:

- be respectful and constructive in technical discussions
- provide actionable, evidence-based feedback
- assume good intent and focus on problem-solving
- keep discussions on-topic and repository-traceable

Unacceptable behavior includes harassment, personal attacks, discrimination,
intimidation, deliberate disruption, or sharing private information without
permission.

## Reporting

Report conduct issues to repository maintainers via private channel.
For security-sensitive incidents, use the process in `SECURITY.md`.

## Enforcement

Maintainers may remove comments, close discussions, or restrict repository
permissions for behavior that violates this Code of Conduct.
